# Horizon Weather App 🌤️

Horizon is a modern weather forecast web application built using HTML, CSS, JavaScript, and Tailwind CSS.  
The application provides real-time weather updates, current location forecasts, and a 5-day extended forecast using the OpenWeatherMap API.

The project focuses on creating a clean, responsive, and visually appealing user experience with dynamic weather backgrounds and glassmorphism UI design.

---

# Features

- Real-time weather updates
- Search weather forecasts by city name
- Get weather details for the current location
- Real-time weather data using OpenWeatherMap API
- 5-day extended weather forecast
- Temperature unit toggle (°C / °F)
- Dynamic weather backgrounds based on weather conditions
- Recently searched cities using Local Storage
- Custom error popup handling
- Extreme temperature alerts
- Responsive design for desktop, tablet, and mobile devices
- Glassmorphism-inspired modern UI

This project uses the OpenWeatherMap API for fetching weather data.

https://openweathermap.org/current

### 5-Day Forecast API
https://openweathermap.org/forecast5

# Functionalities Implemented

## Current Weather Information
Displays:
- Temperature
- Weather Condition
- Humidity
- Wind Speed
- Pressure
- Visibility
- Feels Like Temperature

---

## 5-Day Forecast
Displays:
- Daily weather icon
- Maximum and minimum temperature
- Humidity
- Wind speed

---

## Dynamic Background System
The application changes the background dynamically based on:
- Clear weather
- Rain
- Clouds
- Thunderstorm
- Mist / Fog / Haze

---

## Local Storage Support
Recently searched cities are stored using browser local storage and displayed in a dropdown section.
